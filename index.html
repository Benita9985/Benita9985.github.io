UNIT 3 - HUMAN RESOURCE MANAGEMENT (HRM)

3.1 Introduction to HRM

Human Resource Management (HRM) is the process of managing people in organizations in a structured and thorough manner...

3.2 Nature, Importance & Scope

  • Nature: HRM is both an art and a science. It involves managing people effectively to improve productivity.
  • Importance: Helps in recruitment, training, performance appraisals, etc.
  • Scope: Includes manpower planning, job analysis, compensation, industrial relations, etc.

3.3 Functions of HRM

  1. Procurement Function
  2. Development Function
  3. Compensation Function
  4. Integration Function
  5. Maintenance Function
